# Jogo do número secreto
## Objetivo
Neste jogo, você tem por objetivo adivinhar o número secreto, um número escolhido aleatóriamente pelo computador na menor quantidade de tentativas possíveis.
Diferente de jogos tradicionais de adivinhação, este será totalmente online e será open source (De código aberto), isto é, qualquer desenvolvedor poderá contribuir para melhorar o jogo.
## Como surgiu?
O jogo do número secreto está sendo desemvolvido por Gabriel Ribeiro, durante a formação de lógica de programação da Alura.
No entanto, mesmo após a concluzão do jogo, será mantido aberto para aqueles que quiserem ajudar a melhorar o programa, trazendo muitas novidades e melhorias.
## Tecnologias utilizadas
As tecnologias utilizadas neste software são:
HTML, CSS e JavaScript.
O HTML e o CSS inicialmente serão fornecidos pela Alura, mas, ao decorrer do desenvolvimento de Gabriel Ribeiro, o designe sofrerá mudanças.
Espero que goste do projeto.
